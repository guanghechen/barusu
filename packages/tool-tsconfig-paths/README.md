# Usage

  * Install
    ```shell
    yarn add --dev @barusu/tool-tsconfig-paths
    ```

  * Correct path alias in `*.d.ts`
    ```shell
    yarn resolve-tsconfig-paths [-p tsconfig.json] [--dts ./lib/types]
    ```
