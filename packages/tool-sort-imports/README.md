# Usage

  * Install
    ```shell
    yarn add --dev @barusu/tool-sort-imports
    ```

  * Usage
    ```shell
    npx sort-imports 'src/**/*.ts'
    ```

  * Options
    - `[indent='<indent>']`
    - `[quote='<quote>']`
    - `[max-column='<column limit>']`
    - `[log-level='<debug>']`
